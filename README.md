# Wat is ambitie?

### Een plek waar jij je eigen project gaat maken
Je gaat werken aan een project dat helemaal van jou is. De planning van het project zal een belangrijk onderdeel zijn. Je verdeelt je project over de weken die er nog in deze periode zijn. In die weken maak je je eigen planning. Elke week maak je in de eerste les een planning voor specifiek die week (sprint planning).

### Hoe werkt het?

- Soort SHINE
- 1 dagdeel in de week
- Individueel 
- 1 of 2 verschillende projecten 
- Bepaal je eigen planning
- Sprints (SCRUM)
